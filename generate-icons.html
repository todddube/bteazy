BTEazy Icon Generator

This tool generates icons for the BTEazy extension with a modern cyan-to-purple gradient. The icons feature a download arrow symbol representing easy torrent downloads.

16x16 pixels

How to Use

  1. Click "Download All Icons" to download all three icon files
  2. Create an "icons" folder in the extension directory
  3. Move the downloaded PNG files into the icons folder
  4. Your extension is now ready to use!

Note: You can also create custom icons using any image editor and save them as PNG files with these exact dimensions.
